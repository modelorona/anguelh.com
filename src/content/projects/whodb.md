WhoDB is a lightweight database management tool that combines the simplicity of a chat interface with the power of a full-featured database manager. It supports a wide range of databases including PostgreSQL, MySQL, SQLite3, MongoDB, Redis, MariaDB, and Elasticsearch, all accessible through a single unified interface.
<br /><br />
WhoDB can be deployed as a single Docker container and provides an intuitive web-based UI for exploring schemas, running queries, and managing data across different database types without switching between tools.
