connparse parses database connection strings, DSNs, URLs, file paths, and cloud storage URIs into one safe structured object. It provides a unified API for handling connection strings across a wide range of formats and protocols, making it easy to extract host, port, credentials, and other fields without rolling your own parser for each database or storage backend.
