deptrust is a CLI tool that checks package versions for known vulnerabilities across npm, PyPI, crates.io, Go modules, RubyGems, NuGet, Maven, Packagist, pub.dev, CocoaPods, Hex.pm, Hackage, GitHub Actions, and more. It also runs as an MCP server, making it easy to integrate dependency vetting into AI-assisted workflows and editors.
